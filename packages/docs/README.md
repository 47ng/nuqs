# docs

This is a Next.js application generated with
[Create Next Docs](https://github.com/fuma-nama/next-docs).

Run development server:

```bash
npm run dev
# or
pnpm dev
# or
yarn dev
```

Open http://localhost:3000 with your browser to see the result.
